queryTool

This web application is designed to make it faster to utilize Google and GPT-4 search templates.

It functions as a redirect hub for the resources specified in the navigation menu, adding functionality by completing complex search queries.

How To Use

For GPT, instructions refer to the actual operation you want GPT to complete, e.g. answer this question.

Context refers to the question to be solved, e.g. what's 9 + 10?.

Clicking search will return a completed prompt template, which follows OpenAI best-practices. This allows the model to respond with a higher degree of accuracy.

For non-GPT options, simply type your question and click search.
